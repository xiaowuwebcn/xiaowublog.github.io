歌曲《Just One Last Dance》

发布 : 2019-01-23 分类 : 音乐 浏览 :

这是Sarah Connor与前夫marc terenzi合唱的歌曲，但最终这对明星夫妇的婚姻还是走到了尽头。婚姻破裂后，两人都一度很悲伤。曾经的《love to be loved by you》最终成为了《Just one last dance》。但做不成夫妻，仍然是好朋友。
