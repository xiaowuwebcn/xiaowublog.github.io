通过V2EX发帖调研github针对中国的优化建议与回帖的回答

发布 : 2019-01-14 分类 : 群友 浏览 :

今天在V2EX看到一篇《[调研]除了免费私有库，各位中国开发者对 GitHub 还有什么期待？》一文，看到回帖的人，发现好多好多的不支持github出现汉化的回帖，

今天在V2EX看到一篇《[调研]除了免费私有库，各位中国开发者对 GitHub 还有什么期待？》一文，看到回帖的人，发现好多好多的不支持github出现汉化的回帖，
我们先看看原文：

Github Free 的好消息这两天引得大家的广泛关注，大家还想要更多福利吗？

有幸和 GitHub 的 CEO @Nat Friedman 见过面也聊过多次，他们非常重视中国开发者的使用体验和用户诉求，我很愿意帮他们问问大家。

这里希望和各位开发者一起讨论并收集一下 GitHub 还有什么值得优化的功能或体验（特别是对于中国开发者），我将汇总整理给官方。

先抛砖引玉：

解决 Github Pages 和 Gist 和访问受阻问题（他们说已经尽力了，还在想办法）

提供 Github 站内的同步交流方式（聊天工具：解决 QQ 群、微信群需求）

优化提供 项目的中国本土化捐赠方式（优化二维码捐赠的方式，如微信 /支付宝支付）

大家还有什么好的建议或意见，可以一起讨论，特别是如何更好的服务中国开发者 Github 还要做些什么？优先解决什么问题？

他们官方会有专人跟进，期待 Github 在中国的体验也越变越好~

第 1 条附言 · 3 天前
补充说明：提到 QQ/微信交流群、支付宝 /微信捐钱二维码等等，这些都是经常在中国开发者项目首页见到的内容，如果官方能为全世界的开发者提供同步的站内交流工具和接受捐赠去掉，就可以避免贴非常多不美观的二维码了，这是官方努力的方向。

第 2 条附言 · 1 天前
首先，感谢大家的广泛讨论，这里整理摘取了部分给 Github 的优化建议，我先反馈一波给到官方。
欢迎继续讨论：

解决网络问题，特别是 Clone Releases 在中国的下载速度需要得到优化，AWS s3 常常链接不上

一部分用户希望支持中文界面，另一部分用户反对

支持 Repo、Star 功能需要分组或者 Tag 方式来方便管理和检索

支持 Follow 用户时可以添加备注

提供 站内交流的功能，如：“Chat tools”

提供 捐赠功能，如：“Donate me”

支持 Repo Code 全文检索，中文分词

支持 中文字体代码显示

增加 Github Universe China 线下社区

提供 项目列表树功能，如：Octotree

支持 解码 jpeg 等常见二进制

优化 review 时的 diff 界面滑动对齐界面

增加 Repo Collection 支持，避免用 awesome-xxx 的 repo 方式

支持 Markdown MathJax

GitHub Pages 支持 TLS 1.3

Github Pages 支持百度搜索引擎检索

而我们大天朝的一些回复真的太搞笑了，但是也感觉有很悲催的心理。
