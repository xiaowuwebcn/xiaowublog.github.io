博客是我在网络社交里的最后一片净土

发布 : 2019-06-06 分类 : 随笔 浏览 :

随着年龄的增加你会发现身边的朋友越来越少，从同学关系变成朋友朋友关系的就更少了。

但是随着现在的网络社交迅猛发展，以及社交的粘结性等因素。就像现在的朋友圈以及一些其他的社交APP，把亲朋好友从现实中都搬到了网络上了，但是这样就发现了一些比较别扭的事情。

现在的人都不敢在这些社交上说出自己的真实想法了，所以现在造就了一批“假”的状态，或者是说病态的信息也不为过。

正是因为这样的生活。所以我说博客是自己在网络上的最后一片净土。

什么是个人博客

个人博客是非常小众的，并且是一个完全自主搭建的一个网络上的小窝。并且自己有该博客的所有权。

所有权
所有权

重要的事情说三遍，是所有权。

个人博客的优点与缺点

缺点：一没有异域风情，二没有知识传播，三没有技术输出，四自己耗时搭建和维护，但是看的人非常的少。

优点：完全自主搭建，所有的数据和资料都在自己手里，可以做到自由的上传与下载，或者瞬间销毁，不会在网络上留下只言片语。不像微信，QQ，微博等社交。所有的数据和资料都在他们手里，如果一但有点小事，你连删除资料的权利都没有。

并且你只是有使用权，并没有该账户的所有权

朋友圈、微博和博客的区别

朋友圈和微博这类东西归结于是旅游团，每天都是新鲜的内容，各种小道消息、天朝大事件，但是大多数是走马观花既没有观点又没有沉淀，可能看过一条消息以后得出的最多的结论是“哦，那条消息我在微博上看到了”仅此而已，但是博客相当于自由行，想怎么玩全看自己，可以将你看到的、想到的，都形成文字沉淀下来。

朋友圈和QQ空间自然是不现实了，假如把老村长和王寡妇的事情写在朋友圈，第二天村长就得找你拼命。如果晚上偷偷溜进王寡妇家的是我，我可以在朋友圈以外的地方写出来，并生动形象的描述当晚所发生的事情，别人看到了最多说：“哇，你居然和王寡妇那啥。”“王寡妇技术怎么样？”但如果是放朋友圈呢，第二天远在海外的隔壁老王都能知道这件事。

博客既然是基于生活就得远离生活，不然自己的一丁点儿的事情都让身边的人都知道了，在他们面前就跟没穿衣服一样。

这里的写东西不仅仅是指写博客这样的长文章，也可以是一条简短的心情状态。所以多年以前我就不用QQ空间了，后来又关闭了朋友圈，微博类的应用我用得更多的是 facebook, 一般人你到是来这里看我啊。所以我是把博客和facebook作为朋友圈外的一方净土，可以肆无忌惮的吐槽。
