转载：有意思的国内外著名互联网企业的名字文化

发布 : 2018-12-28 分类 : 转载 浏览 :

前不久，马云在2018阿里巴巴校友见面会上详细解读了阿里巴巴、天猫、平头哥等名字的由来。不得不说，互联网公司的名字文化真的十分有意思，在这里小编搜刮了国内外著名互联网企业的取名来历，让大家一观。

阿里巴巴

阿里巴巴就是芝麻开门的意思，表示为全世界所有的小企业open Sesame，让天下没有难做的生意。阿里巴巴这个名字是马云在美国一家餐厅吃饭时想到的，他发现阿里巴巴的故事被全世界的人所熟知，且不同语种的发音也近乎一致，因此确定了这个名字。不过，当时“阿里巴巴”域名已被一个加拿大人买下，为此他还拿出了1万美元买回了这个域名。

百度

百度来源于中国宋朝词人辛弃疾《青玉案·元夕》的诗句：“众里寻她千百度，那人却在灯火阑珊处”。百度要做最懂中国人的搜索引擎，所以最初就想带入中国传统文化。而李彦宏从小就喜欢唐诗宋词，在思索中灵光乍现，突然冒出辛弃疾的这句词。确实，“百度”两字将李彦宏想要表达的东西都涵盖了，中国文化博大精深呀。

腾讯

腾讯的来源，一是马化腾的名字里有个“腾”字，二是“腾”有腾飞、发达之意。“讯”是因为老东家润讯对马化腾的影响。而英文命名Tencent是参考著名通讯公司朗讯(lucent)而起的。据说Tencent的英文名被香港人称为“十分钱”(tencent)。别说，这很贴切形象，因为那时的腾讯是家短信内容提供商公司，用户发短信腾讯收费一毛一条，正好十分钱。

华为

华为是创始人任正非随意取的，因为当时墙上贴着“中华有为”的标语。在烦恼公司名字许久之后，任正非直接拍板取名为华为。据说因为“华为”的发音不响亮，十几年来华为公司内部一直在争议要不要改掉这个名字。

苹果

苹果名字的来源有很多说法，有人说是致敬牛顿，图灵，图灵破译了战争时期德国的电码，最后却食用浸过氰化物的苹果自杀了。不过最后乔布斯自己辟谣说并没有考虑到这一点。真正的原因是乔布斯读大学的时候，有一位好友罗伯特的叔父经营着一个很大的苹果园，那年夏天乔布斯去苹果园打工，突然来了灵感。看吧，当一个公司出名之后，人们总会把它想得很有深意。

Google

google最开始其实是 googol，指的是10的100次幂，写出的形式为数字1后跟100个零，用来代表互联网上可以获得海量的资源。不过，在互联网域名注册数据库里搜索时，词被打成了“google .com”，而它刚好可以使用，就被定下了。毕竟，名字想得再好都比不上能够使用来得好。

Microsoft

Microsoft 这个名字真的很科学，它首次出现于盖茨1975年给微软另一创始人保罗·艾伦的信中。表示公司将专注于MICROcomputerSOFT-ware(微型计算机 软件)，这像个混合“微型计算机”和“软件”而成的单词。最开始其实叫Mi-cro-Soft，后来把中间的“-”去掉了。

Twitter

Twitter的雏形是“twitch(抽动)”，希望表达一种世界各地都在嘁嘁喳喳谈论的感觉，但它太抽象。所以选择了字典中相近的字词“Twitter”，一种鸟叫声，创始人认为鸟叫是短、频、快的，符合网站的内涵。

Facebook

Facebook的灵感来源于同学录，某年的秋季，扎克伯格和其他新生一样入学时，他收到了一本埃克塞特中学同学录——“图片地址花名册”，同学们都称之为“The Facebook”，Facebook的名称即来源于此。

NOKIA

诺基亚(Nokia)其实是生活在芬兰的一种长着黑色长毛与利爪的野兽。由于这种野兽主要生活在芬兰境内一条河的两岸，因此，人们以这个怪兽的名字来命名这条河——诺基亚河。诺基亚公司最开始就是设立在这条河边。

搜狐

搜狐最开始叫“搜乎”，很浅显易懂。但市场部经理陈剑锋认为搜乎并不够好，因此对比当时的互联网巨头yahoo，改为sohu搜狐。因为陈剑锋感觉当时的张朝阳非常像“狐狸”，很聪明、也很敏感，也令人捉摸不透。

Yahoo

Yahoo来源于1726年出版的英国讽刺小说《格列佛游记》，这是一个全新的词，代表一群没有受过教育的野人。据杨致远后来回忆，他们把自己的公司取名为Yahoo，也是带着一点恶搞的意味。

网易

丁磊的解释是觉得当时使用网络的人很少，因为中文网站很少，上网费很高，上网是件很让人生畏的事。而网易要做的就是使这种情况发生改变，变得容易些——网易。

新浪

sina一词源于拉丁文的中国——sino，在英语中也经常应用，如sino-american是中美关系的意思，sina在古印度和古希腊是对中国的敬称。sina的中文名称新浪是当时的总裁王志东起的，这表达了新浪希望自己成为中华区最大门户的决心。

豆瓣

豆瓣的创始人阿北曾经在北京朝阳门旁边的豆瓣胡同住过，还在胡同附近的星巴克写出了豆瓣网最初的源代码。其实这就是豆瓣的名字来历，是不是很任性。

小米

最初小米公司在筹办的时候有过很多提议，比如玄德科技，因为小米公司是在北京五道口的逐鹿茶馆里的玄德厅筹办的。但有一天小米科技创业团队最早的14个人围在一起喝了一碗小米粥，据说这碗小米粥是联合创始人黎万强的父亲早上5点钟起来熬好送到公司的。而小米是中国人主要的食粮，大家都离不开，所以决定叫小米公司。

摩拜单车

摩拜是天使投资人李斌帮忙取的，把mobile和bike拼起来叫mobike，中文摩拜是“mobike”的音译，就像顶礼膜拜的谐音。按摩拜CTO夏一平说，最初创业团队想给这款单车起名叫“丁丁单车”，这样听起来更上口、还接地气。

企业的名字特别容易被误读或是过分解读，虽然大部分还是包含了深刻的意义，但依然有些比较随意。而这些看起来好像很随意的名字会随着企业的成功不断升华变得意义非凡。你最喜欢哪个名字呢?
