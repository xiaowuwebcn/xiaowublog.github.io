转载：利用代码托管平台与Hexo搭建个人博客，这个是小吴博客做的视频教程，感觉还不错，转载一篇哈，毕竟自己录制和制作的视频教程，是真心不容易的。
视频地址：https://www.bilibili.com/video/av36175038

搭建本地环境

安装Git
安装node.js

安装和初始化博客框架hexo

安装hexo博客框架 npm install hexo -g
hexo是否安装成功 hexo -v
初始化本地文件夹 hexo init
安装组件 npm install
本地生成并预览 localhost:4000 Ctrl+C停止预览 hexo s -g
清除本地缓存 hexo clean
选择代码托管平台部署博客
– 常见的代码托管平台 github coding gitlab 等

注册账号

创建博客项目
将git绑定用户名和邮箱
git config –global user.name “name”
git config –global user.email “xxx@qq.com“

绑定SSH

生成SSH文件 ssh-keygen -t rsa -C “xxx@qq.com”
添加SSH秘钥到代码托管平台上
测试添加ssh是否成功 ssh -T git@github.com
– hexo配置部署

deploy: 
 type: git
repo: github@github.com:xx/xxx.github.io.git

安装部署插件 npm install hexo-deployer-git –save

部署上传 hexo d -g

github用户名.github.io访问博客
