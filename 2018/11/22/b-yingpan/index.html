买到假硬盘竟然追回9136元？退一赔三315投诉及民事起诉全流程

发布 : 2018-11-22 分类 : 随笔 浏览 :

双11刚过，想必大家的手已经剁没了。买到称心如意的商品自然值得欣喜，但如果遇到假货怎么办呢？本次视频soso带你从固定证据开始，把平台调解、315投诉、民事起诉整个走一遍！

原帖：https://www.bilibili.com/video/av35862750/

网友评论：

  1. 对B站部分的年轻用户来说，是很好的普法教育，有利于市场规范化，可以有效屏蔽商品标低价结果货不对板的情况。至于有些人基于“得饶人处且饶人”的想法，对SOSO进行道德上的批判，我只想说，第一你没有上帝视角，希望能够客观就事论事，不要因为听见卖家“可怜”的申辩就断定卖家弱势；第二，伸张自己的正当法律权益是每个人权力，尊重法律而不是基于自己主观判断的应该不应该；第三，不要以为放过这一家也没什么，如果每个人对这种情况，光靠妥协，抱着忍一时风平浪静的想法，那最终的成本还是转嫁到你们每个人头上。我在生活中经常扮演的是卖方角色，对于那些价格便宜明显异常的东西，我和消费者都心知肚明是怎么回事，也碰上过所谓的“职业打假人”，但迫于同业竞争，一部分人的不规矩扰乱了整个市场秩序，不跟进的下场就是被这些劣币驱逐。我希望从小就能通过网络看到世界的年轻人们能够拥有这样的意识，不要再无端承受这些耍小聪明的奸商带来额外的时间和金钱上的成本。

  2. 从这个视频可以看出，普法和尊法的教育还需要进一步普及。目前对于消费者的保障已有了很大进步。但是仍然可以发现维权难，卖家协商，平台协商，消协举报，到最后诉诸法律，同时还要主动保存证据，包括网页截图，交易截图，拆封视频电话录音和交流截图等。
    若不是up主职业相对时间更加自由宽松，硬盘价值较高同时up主有主动留证据的意识的话，最后的结果可能就不会这么美好了。
    感谢up主使用自己的大量精力和时间来维护更多人的利益，从卖家的不承认与推诿，平台希望大事化小和对帮助买家维权的无能为力，也能从侧面反映现在假货风险成本低，维权成本高的现实。

  3. 法律是道德的最低底线，如果连法律都不懂的话，那就是没有底线。玩蛋去吧。

  4. 京东1399的希捷8t比淘宝稳多了。
    再然后这个假货问题，hp仅仅因为低配置笔记本电脑的广告用了高配置型号的宣传图片，然后假一赔三了。
    合法权益就是合法的，自己不用不要阻拦别人用。
